CIFAR-10 and CIFAR-100

The CIFAR-10 and CIFAR-100 datasets consist of colored images of size 32x32 pixels. These datasets are widely used for benchmarking image classification algorithms due to their standardized format and balanced class distribution.

CIFAR-10 Dataset

CIFAR-10 contains 60,000 color images across 10 classes, with 6,000 images per class. The training set contains 50,000 images, while the testing set contains 10,000 images. The classes include airplanes, automobiles, birds, cats, deer, dogs, frogs, horses, ships, and trucks.

CIFAR-100 Dataset

CIFAR-100 contains 100 classes with 600 images per class. These are grouped into 20 superclasses, providing a more challenging classification task with finer-grained distinctions between categories.
